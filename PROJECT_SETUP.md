# Fabulous Boost - Project Setup

This is a cloned project from Boostelix with the new name "Fabulous Boost".

## Quick Start

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Environment Setup**
   - Copy `.env.example` to `.env` if it exists
   - Update `VITE_APP_BASE_URL` to point to your backend API

3. **Run Development Server**
   ```bash
   npm run dev
   ```

## Project Structure

- Frontend: React + Vite
- Backend: Laravel API (fabulousboostbackend)

## Important Notes

- This project is independent from Boostelix
- Update all environment variables for your new setup
- Update API endpoints if using a different backend URL
- Update branding in `index.html` and throughout the app
